技術與美學

1.Supabase：

作為專案的資料持久層與後端管理平台。利用關聯式資料庫儲存飲水機的基礎數據、地理座標以及開發里程碑。透過資料庫的 API 接口與行級安全（RLS）權限控制，確保前端網頁能即時且安全地讀取最新資料。

2.Python (粉絲數爬蟲)：

專案的自動化數據採集引擎。利用自動化測試工具（如 Playwright）模擬瀏覽器行為，定期存取社群平台公開頁面以擷取各帳號的最新粉絲數量，並將數據同步至資料庫中。

3.Vercel:

託管基於原生 JavaScript 構建的單頁應用程式 (SPA)，確保網頁在全臺各地具備低延遲的存取體驗。 前端整合了 Chart.js 進行品牌市佔率與趨勢的圖表可視化，並利用 Font Awesome 提供介面標識系統。 該平台承載了縣市總數統計、個人粉絲排行榜以及社團資訊展示等視圖切換邏輯，並負責透過 API 請求即時展現資料庫中的最新數據。

4.GitHub (版本控制與 CI/CD 自動化架構):

負責程式碼的版本管理與協同開發。利用其內建的 GitHub Actions 或連動機制驅動 持續整合與持續部署 (CI/CD) 流程。 當開發者將最新的爬蟲代碼 or 前端修正推送至儲存庫時，GitHub 會自動觸發 Vercel 的部署任務，確保本機開發 environment 與雲端運算資源的一致性，並簡化了跨設備開發所需的環境配置成本。

5.Apple 極簡主義 (UI/UX 設計規範):

定義了專案的視覺呈現規範，強調高訊號雜訊比 (Signal-to-noise ratio) 與大量留白設計。 色彩系統採用 #ffffff 與 #fafafa 作為背景基調，搭配 #1a1a1a 深灰色文字確保閱讀對比度。 版面配置採用彈性佈局（如寬螢幕下的雙欄並排與手機端的垂直堆疊），並優先調用系統預設字體層級，旨在為使用者提供穩定、現代且專注於數據本身的專業儀表板體驗。
